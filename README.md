# install
`npm install -g set-game`

# play a game:
`set-game play --html`

# check a board
`set-game solve`

type cards in number, fill, color, shape order, separated by spaces or newlines,
so for 1 open red diamond:
```
1ord
^D
```

Send an EOF (`^D` or `^Z` on Windows) when done
